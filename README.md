## 📋 Sobre o Projeto

**Spa Lisiane Garroni** 
Website profissional desenvolvido para o Spa Lisiane Garroni, oferecendo uma experiência moderna para clientes conhecerem os serviços e realizarem agendamentos. O projeto utiliza arquitetura moderna com frontend responsivo em React e backend robusto com Node.js + Express.

### ✨ Features Principais

- 📱 **Design 100% Responsivo** - Mobile, Tablet e Desktop
- 💬 **Integração WhatsApp** - Botão flutuante com conversão
- 📸 **Galeria Interativa** - Carrossel de antes e depois
- 📅 **Formulário de Agendamento** - Integrado com backend
- 🎨 **Design Profissional** - Tema bronze/dourado elegante
- ⚡ **Performance Otimizada** - React + Vite
- 🔒 **Segurança** - Validação frontend e backend

## 🛠️ Como Instalar

### Pré-requisitos
- Node.js (v16+)
- npm ou yarn

### Frontend

```bash
# Clonar repositório
git clone https://github.com/CGarroni/Spa-LisianeGarroni.git
cd Spa-LisianeGarroni/frontend

# Instalar dependências
npm install

# Iniciar desenvolvimento
npm run dev

# Build para produção
npm run build
```

### Backend (Próxima Etapa)

```bash
# Entrar na pasta backend
cd ../backend

# Instalar dependências
npm install

# Iniciar servidor
npm run dev
```
